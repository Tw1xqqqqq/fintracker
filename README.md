# FinTracker Desktop

Desktop-приложение для план-факт учета личных финансов, прогнозирования баланса и анализа расходов.

## Стек

- Tauri 2.x
- TypeScript
- React
- SQLite через `tauri-plugin-sql`
- TanStack Table
- Recharts

Что сделали 15.07.2026
- Обновили сайдбар под макет из фигмы
- Подтянули базовые элементы под макет: кнопки, инпуты, select, карточки, модалки
- Сделали сверку баланса - ввод фактического остатка, отчёт о расхождениях, корректирующая операция 
- Написали функционал спец-операции (Пока доделываем)
- Покрыли расчёты 2–3 юнит-тестами на ключевые случаи.
