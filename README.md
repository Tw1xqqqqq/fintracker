# FinTracker Desktop

Кросс-платформенное desktop-приложение для план-факт учёта личных финансов: еженедельное бюджетирование, автопересчёт цепочки баланса, подсветка кассовых разрывов и импорт банковских выписок. Концепция выросла из Excel-прототипа «Мой бюджет» с недельными колонками.

## Возможности

- **Бюджет на год** — таблица «статья × неделя» с режимами «План» и «Факт»: редактируемые плановые суммы, факт с дельтой «к плану», строки «Баланс» и «Итоговый баланс» с переносом остатка между неделями.
- **Кассовые разрывы** — недели, где баланс уходит в минус, подсвечиваются в таблице с предупреждением.
- **Операции** — журнал с группировкой по дням, поиском, фильтрами (период, тип, статус, счёт, категория) и карточками итогов; доходы, расходы и переводы между счетами (две связанные операции без задвоения баланса).
- **Регулярные платежи** — правила повторения (день/неделя/месяц/год, дни недели, «7-го числа» или «первый вторник», окончание по дате или числу повторов) автоматически раскладываются в план; операции текущей недели предлагаются к подтверждению в факт.
- **Импорт CSV-выписки** — кодировки UTF-8/Windows-1251, автоопределение разделителя и колонок, автокатегоризация по описанию, экран подтверждения, защита от дублей при повторном импорте.
- **Экспорт операций в CSV** — выгрузка журнала для Excel.
- **Счета и статьи** — справочники с текущими остатками по каждому счёту (стартовый баланс + фактические операции, переводы по ногам).

## Стек

- [Tauri 2](https://tauri.app/) (Rust) + WebView
- TypeScript + React 18 + Vite
- SQLite через `tauri-plugin-sql` (миграции версионируются в Rust)
- Vitest — 77 юнит-тестов на расчётный движок и парсер импорта

## Запуск в разработке

Требования: Node.js 20+, Rust (toolchain MSVC на Windows), WebView2.

```bash
npm install
npm run tauri dev
```

При первом запуске применяются миграции БД и открывается онбординг (создание счёта и даты начала учёта). Файл базы: `%APPDATA%/com.fintracker.desktop/fintracker.db`.

## Сборка установщика

```bash
npm run tauri build
```

Артефакты: `src-tauri/target/release/bundle/msi/*.msi` и `.../nsis/*-setup.exe`.

npm test
