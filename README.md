# FinTracker Desktop

Desktop-приложение для план-факт учета личных финансов, прогнозирования баланса и анализа расходов.

## Стек

- Tauri 2.x
- TypeScript
- React
- SQLite через `tauri-plugin-sql`
- TanStack Table
- Recharts

Что сделали 13.07.2026
- Сделали экран «Дашборд» с загрузкой и обработкой ошибок.
- Добавили карточки сверху: текущий баланс, план и факт за период, число кассовых разрывов
- Нарисовали график: линия баланса по неделям, столбцы доходов/расходов
- Подключили экран вместо заглушки «Дашборд».
- Покрыли расчёты 2–3 юнит-тестами на ключевые случаи.
