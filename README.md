# FinTracker Desktop

Desktop-приложение для план-факт учета личных финансов, прогнозирования баланса и анализа расходов.

## Стек

- Tauri 2.x
- TypeScript
- React
- SQLite через `tauri-plugin-sql`
- TanStack Table
- Recharts

Что сделали 14.07.2026
- Реализовали редактирование ячеек и сохранение плана в БД с пересчётом баланса
- Подключили раздел «План» вместо заглушки
- Построили таблицу планирования 
- Добавили мастер финансового года: выбор года и дат начала/конца
- Добавлены регулярные платежи
- Покрыли расчёты 2–3 юнит-тестами на ключевые случаи.
